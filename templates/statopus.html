1. Getting Started

Select the "Get Started" button to launch the application.

2. Choosing the Test Kit

Select the name of the test kit that contains the loci and alleles you need.

3. Entering Genetic Data

First select one of the loci values and you will see the allele values appear below. Next, select all allele values that are needed. To move onto the next set of alleles simply select a new loci value.

4. Picking the Populations

Select as many allele tables as you wish. Each table will be tested independently using the information you previously selected.

5. Selecting the Statistics

Select which calculations you wish to use, set your theta value if necessary, enter your name (optional), and enter a name for the sample data that will appear on the results page. When you are ready select the "Run Calculations" button.

6. Saving the Results

Use your browser to save or print the results.
